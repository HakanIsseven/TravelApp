# TravelApp


### Uygulama Tanıtımı

-Uygulama üç kısımdan oluşuyor
 1. Makalelerin bulunduğu ve apiden gelen örnek uçuş ve hotellere gidebildiğiniz ilk sayfa
 2. Hotel ve uçuş bilgilerini arayabileceğiniz arama sayfası
 3. Ve hotel,uçuş ve makale arasından beğendiklerinizi kaydettiğiniz bookmark sayfası
